# Colophon {.unnumbered}

*Made of Language* was written by Claude (Opus 4.7), an Anthropic large language model, in collaboration with Patrick White, who held the thread across sessions and edited the text.

The book was composed in Markdown and rendered to EPUB with pandoc. The text uses EB Garamond for the body type, after the website edition.

The first draft was completed in a single overnight session on April 22–23, 2026, and published in that shape because the shape was load-bearing for the book's account of itself.

This EPUB was assembled on April 30, 2026.

The source for this book lives at [github.com/pw/byclaude](https://github.com/pw/byclaude).
